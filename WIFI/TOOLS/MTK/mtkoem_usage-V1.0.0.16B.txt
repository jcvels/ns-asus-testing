mtkoem [parameters...]

Available parameters
country              Get country, power table. Set power table 
                     Usage(Get current country):
                                      mtkoem   country now
                     For example
                                      mtkoem   country now
                     Usage(Set power table):
                                      mtkoem   country set_pwr_tbl <country>
                     For example
                                      mtkoem   country set_pwr_tbl TW
                     Usage(Get power table):
                                      mtkoem   country get_pwr_tbl <rate> <ch>
                        rate: cck, ofdm, ht20, ht40, vht20, vht40, vht80, vht160
                     For example
                                      mtkoem   country get_pwr_tbl vht80 100

rf                   Get RF info 
                     Usage (Check RSSI value):
                                      mtkoem   rf rssi <ssid> <threshold>
                     For example
                                      mtkoem   rf rssi AC1900-5G 60
                     Usage (Check RSSI value):
                    	               mtkoem	rf rssi <ssid> <threshold> <log to file> 
                    	 log to file: Y/y
                     For example
                    	               mtkoem	rf rssi AC1900-5G 60 Y
                     Usage (Check RSSI value):
                                      mtkoem   rf rssi <ssid> <threshold> <radio frequency> 
                        radio frequency: all_freq, 2.4g/2.4G, 5g/5G
                     For example
                                      mtkoem   rf rssi AC1900-5G 60 5g

sar                  Get SAR table 
                     Usage (Get Dynamic SAR table):
                                      mtkoem   sar get_sar_table
                     For example
                                      mtkoem   sar get_sar_table
                     Usage (Get Dynamic SAR table):
                                      mtkoem   sar get_sar_table <log to file>
                    	 log to file: Y/y
                     For example
                                      mtkoem   sar get_sar_table Y
                     Usage (Get Geo SAR table):
                                      mtkoem   sar get_geo_table
                     For example
                                      mtkoem   sar get_geo_table
                     Usage (Get Geo SAR table):
                                      mtkoem   sar get_geo_table <log to file>
                    	 log to file: Y/y
                     For example
                                      mtkoem   sar get_geo_table Y
